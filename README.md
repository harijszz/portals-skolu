# Skolu Portāls

Skolēnu atzīmju uzskaites un vidējo svērto aprēķinu tīmekļa aplikācija.

## Sākotnējā ideja

Projekts paredzēts, lai skolēns varētu:
- Pievienot un pārvaldīt savus mācību priekšmetus
- Katram priekšmetam pievienot atzīmes ar vērtību (1–10) un svaru
- Automātiski aprēķināt vidējo svērto atzīmi katram priekšmetam
- Redzēt, vai priekšmets ir nokārtots (vidējā atzīme ≥ minimālā sekmju robeža)
- Izmantot "what-if" kalkulatoru, lai noskaidrotu, kāda atzīme nepieciešama sekmīgam vērtējumam

Aplikācija veidota bez autentifikācijas — to var lietot lokāli kā personīgo atzīmju menedžeri.

## Tehnoloģijas

- Laravel 13
- SQLite
- Tailwind CSS
